# Issue Template

## 🐛 Bug Report

**Description**
- Clear description of the bug.

**Steps to Reproduce**
1. Trigger Airflow DAG
2. Run ingestion script
3. Observe error in logs

**Expected Behavior**
- What should have happened?

**Actual Behavior**
- What actually happened?

**Logs / Screenshots**
- Paste relevant error logs or screenshots.

**Environment**
- OS: [e.g., Ubuntu 22.04 / Windows 11]
- Docker version: [e.g., 24.0.5]
- Airflow/dbt/Postgres versions

---

## 🚀 Feature Request

**Summary**
- Short description of the feature.

**Motivation**
- Why is this feature needed?

**Proposed Solution**
- How should it work?

**Alternatives**
- Any other approaches considered?

## 📋 Checklist Before Submitting
- [ ] I have searched existing issues
- [ ] I have reproduced the bug on the latest `main`
- [ ] I have included logs/screenshots if relevant
- [ ] I have described environment details
